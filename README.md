Contains data analysis for project regarding smartphones and fertility rates. Utilizes NFHS-3 to NFHS-6 data (from drpakhare/NFHS-6-v1.0.0.zip)

Contains 5 figures spanning NFHS-3 through NFHS-6. 
Figure 1: Parallel Decline in Fertility Rates vs. Growth in Mobile Access in select states (India, Kerala, Tamil Nadu, Bihar, Uttar Pradesh, Maharashtra)
Figure 2: Scatter Plot of State-level TFR vs. mobile penetration by NFHS installment
Figure 3: Scatter Plot of Within state changes with 33 states
Figure 4: Choropleth Map of NFHS-6 TFR and mobile
Figure 5: Pairwise correlations between TFR, Mobile, Schooling 10+, FP modern, Electricity, Internet, HH decisions, Bank account

Along with other models/regressions.
Model 1: Pooled OLS (wave FE, state-clustered SEs)
Model 2: State FE + Wave FE (clustered by state)
Model 3: Robustness
